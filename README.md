🧮 Calculadora em Python

Este projeto consiste em uma calculadora desenvolvida na linguagem Python, utilizando funções, listas, estruturas de repetição(While e For) e estruturas de condicionais (If, elif e else).

A aplicação foi criada com o objetivo de praticar lógica de programação e construção de sistemas interativos via terminal utilizando as funções oferecidas pelo Python.

📌 Funcionalidades do sistema

A calculadora possui um menu interativo que permite ao usuário navegar e escolher entre diversas operações livremente:

1.Adição
2.Subtração
3.Multiplicação
4.Divisão
5.Potência
6.Raiz quadrada
7.Visualização do histórico de cálculos
8.Encerramento do programa

💡 Diferencial do projeto

Um dos principais pontos da calculadora é a utilização de listas para armazenar o histórico das operações realizadas.

Diferente de uma calculadora simples, que exibe apenas o resultado, esse sistema que criei, mostra a operação completa junto com o resultado, por exemplo:

2 + 3 = 5

Essa operação fica armazenada dentro da lista correspondente (por exemplo, essac conta de "2 + 3 = 5, vai ficar armazenada numa lista), tornando a experiência mais clara, organizada e facilitando a visualização dos cálculos realizados.

🔄 Funcionamento

O sistema utiliza um laço de repetição (while) para manter o programa em execução contínua, permitindo que o usuário realize quantas operações desejar.

O programa é encerrado apenas quando o usuário seleciona a opção "8", garantindo maior interatividade e controle durante o uso.

🛠️ Tecnologias utilizadas
-Python
-Estruturas condicionais (if, elif, else)
-Laço de repetição (whilen, for)
-Listas
-Função enumerate

🚀 Objetivo

Este projeto foi desenvolvido com foco no aprendizado de lógica de programação, arquitetura de código e interação com o usuário, podendo melhorar o ritmo de projetos criados.

🏁 Resultado

O resultado é uma calculadora funcional, organizada e com um diferencial importante: o histórico detalhado das operações, tornando o projeto mais completo e próximo de uma aplicação real.

👨‍💻 Considerações finais

Durante o desenvolvimento, o projeto evoluiu além da ideia inicial de uma calculadora simples. Novas funcionalidades foram sendo incrementadas ao longo do processo, tornando a aplicação mais completa e estruturada.
