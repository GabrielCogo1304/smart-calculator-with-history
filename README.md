# smart-calculator-with-history
Calculadora em Python com menu interativo e histórico de operações, desenvolvida para prática de lógica de programação.

🧮 Calculadora em Python

Este projeto consiste em uma calculadora desenvolvida na linguagem Python, utilizando funções, listas e estruturas de repetição.

A aplicação foi criada com o objetivo de praticar lógica de programação e construção de sistemas interativos via terminal.

📌 Funcionalidades do sistema

A calculadora possui um menu interativo que permite ao usuário navegar e escolher entre diversas operações:

Adição
Subtração
Multiplicação
Divisão (com verificação de divisão por zero)
Potência
Raiz quadrada
Visualização do histórico de cálculos
Encerramento do programa

💡 Diferencial do projeto

Um dos principais pontos da calculadora é a utilização de listas para armazenar o histórico das operações realizadas.

Diferente de uma calculadora simples, que exibe apenas o resultado, este sistema mostra a operação completa junto com o resultado, por exemplo:

2 + 3 = 5

Essa operação fica armazenada dentro da lista correspondente (por exemplo, a lista de adição), tornando a experiência mais clara, organizada e facilitando a visualização dos cálculos realizados.

🔄 Funcionamento

O sistema utiliza um laço de repetição (while) para manter o programa em execução contínua, permitindo que o usuário realize quantas operações desejar.

O programa é encerrado apenas quando o usuário seleciona a opção "8", garantindo maior interatividade e controle durante o uso.

🛠️ Tecnologias utilizadas
Python
Estruturas condicionais (if, elif, else)
Laço de repetição (while)
Listas
Função enumerate

🚀 Objetivo

Este projeto foi desenvolvido com foco no aprendizado de lógica de programação, organização de código e interação com o usuário, sendo um passo importante na evolução para projetos mais complexos na área de desenvolvimento de software.

🏁 Resultado

O resultado é uma calculadora funcional, organizada e com um diferencial importante: o histórico detalhado das operações, tornando o projeto mais completo e próximo de uma aplicação real.

👨‍💻 Considerações finais

Durante o desenvolvimento, o projeto evoluiu além da ideia inicial de uma calculadora simples. Novas funcionalidades foram sendo adicionadas ao longo do processo, tornando a aplicação mais completa e estruturada.

Esse processo contribuiu diretamente para o desenvolvimento do raciocínio lógico e da capacidade de melhorar um sistema de forma incremental.
